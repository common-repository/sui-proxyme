== Description ==
Sui-ProxyMe is a simple plugin that to show the pictures by Proxy.
It mean that the pictures which are from picasa, flickr or any other site that have been blocked in your country.

== Installation ==

1. Extract all the files.
1. Upload the folder `sui-proxyme` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

If you want to change something, you should to look `config.php` file
